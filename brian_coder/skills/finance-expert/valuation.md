# Valuation Methods

## DCF (Discounted Cash Flow)
```
Enterprise Value = Σ(FCF_t / (1 + WACC)^t) + Terminal Value / (1 + WACC)^n
Terminal Value = FCF_final × (1 + g) / (WACC - g)
```

Steps: Project FCF (5yr) → Calculate WACC → Terminal Value → Discount to PV

## WACC
```
WACC = (E/V × Re) + (D/V × Rd × (1 - Tc))
Re (CAPM) = Rf + β × (Rm - Rf)
```

## Key Ratios
- **Liquidity**: Current Ratio, Quick Ratio
- **Profitability**: Gross/Operating/Net Margin, ROA, ROE
- **Leverage**: D/E, Interest Coverage
- **Valuation**: P/E, P/B, EV/EBITDA

## Sensitivity Analysis
항상 WACC ± 1-2%와 terminal growth ± 0.5% 범위에서 민감도 테스트.
