## Overview

Parameterized N-bit up/down counter with synchronous reset, count enable, parallel load, and overflow detection.

The counter supports four operating modes selected by control inputs with fixed priority:

| Priority | Mode | Condition | Behavior |
|----------|------|-----------|----------|
| 1 (highest) | Synchronous Reset | `rst_n == 0` | `count_out ← 0`, `overflow ← 0` |
| 2 | Parallel Load | `rst_n == 1 && load == 1` | `count_out ← data_in`, `overflow ← 0` |
| 3 | Count Up | `rst_n == 1 && load == 0 && en == 1 && up_down == 0` | `count_out ← count_out + 1` |
| 3 | Count Down | `rst_n == 1 && load == 0 && en == 1 && up_down == 1` | `count_out ← count_out - 1` |
| 4 (lowest) | Hold | `rst_n == 1 && load == 0 && en == 0` | `count_out` unchanged, `overflow ← 0` |

**Overflow Detection**: When counting up at maximum value or counting down at zero, `count_out` wraps around and `overflow` pulses high for one clock cycle.
