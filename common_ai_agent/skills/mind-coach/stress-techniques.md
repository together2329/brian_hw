# Stress & Anxiety Techniques

## Immediate Relief (2-5 minutes)

**4-7-8 Breathing**: 코로 4초 흡입 → 7초 멈춤 → 입으로 8초 내쉼 → 4회 반복
**Box Breathing**: 4초 흡입 → 4초 멈춤 → 4초 내쉼 → 4초 멈춤 → 반복
**5-4-3-2-1 Grounding**: 보기 5개, 만지기 4개, 듣기 3개, 냄새 2개, 맛 1개

## Cognitive Reframing (CBT)

흔한 인지 왜곡과 재구성:
- **흑백 논리**: "완벽 아니면 실패" → "부분적 성공도 가치"
- **과잉 일반화**: "항상 실패해" → "이번에서 배울 점이 있다"
- **재앙화**: "다 끝났어" → "실수는 수정 가능하다"
- **감정적 추론**: "불안하니까 잘못될 거야" → "느낌 ≠ 사실"

## Thought Record
1. 상황 → 2. 자동적 사고 → 3. 감정 (0-100) → 4. 증거/반증 → 5. 대안적 사고 → 6. 재평가

## Panic Attack Protocol
1. 안전한 장소 이동 2. 5-4-3-2-1 그라운딩 3. Box Breathing 4. "이것은 지나갈 것이다" 반복 5. 차가운 물로 얼굴 씻기 6. 신뢰하는 사람에게 연락
