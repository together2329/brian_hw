# Todo Execution Rules

이 파일의 내용은 각 태스크 시작 시 AI에게 자동으로 주입됩니다.
프로젝트 루트 또는 ~/.common_ai_agent/.TODO_RULE.md 에 위치시키세요.

## 실행 규칙

- 태스크를 시작하기 전에 반드시 관련 파일을 먼저 읽고 파악하라
- 변경 전 현재 코드의 동작을 이해하라
- 한 번에 하나의 파일씩 수정하고 확인하라
- 수정 후 관련 기능이 정상 동작하는지 검증하라
- 완료 판단 기준(criteria)이 있으면 모두 충족했는지 확인 후 completed 처리

## 완료 기준

- 기능이 의도대로 동작함을 확인
- 기존 동작이 깨지지 않음을 확인
- 필요한 경우 테스트 실행
