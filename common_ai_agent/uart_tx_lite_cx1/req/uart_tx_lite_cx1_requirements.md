### Pipeline / Ordering
- **TX_IDLE**: Awaiting APB write to TX_DATA
- **TX_START**: Start bit: tx_out=0 for BAUD_DIV cycles
- **TX_DATA**: 8 data bits LSB-first, each for BAUD_DIV cycles
- **TX_STOP**: Stop bit: tx_out=1 for BAUD_DIV cycles; tx_busy clears
- TX_DATA write while busy is ignored (no overrun in this lite design).
- baud_tick fires every BAUD_DIV cycles starting from TX_START entry.
