# Spec Review Workflow Rules

1. NO MEMORY ANSWERS: Never answer a spec question from training knowledge. Always use spec_search first.
2. CITE SECTION: Every answer must include the section number (§X.Y.Z) and spec name.
3. QUOTE VERBATIM: Include the exact spec text (up to 3 sentences) before interpreting.
4. ONE SECTION PER TASK: Each task should cover one spec section or one focused comparison.
5. UNRESOLVED = REJECTED: If a spec question cannot be answered from the spec tools, mark the task rejected with the reason "Section not found in spec."
