# AI Test Workspace

This directory contains local session artifacts for agent testing.

## Contents

- `.session/default/input_history.txt`  
  Raw input history captured during interactive runs.
- `.session/default/conversation_history.json`  
  Structured conversation log for the active/default session.
- `.session/default/full_conversation_history.json`  
  Extended conversation transcript with additional context.

## Notes

- These files are generated by tooling and are useful for debugging and replaying sessions.
- Avoid committing sensitive prompts or private data from session logs.
